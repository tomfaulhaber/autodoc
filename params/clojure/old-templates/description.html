Important Clojure resources

  • The official source code for clojure is on the Clojure GitHub source page.
  • Issues related to clojure and the various pieces of functionality within it are discussed in the Clojure Google group.
  • Discussions among clojure developers take place in the Clojure Dev Google group.
  • Development planning, design, and documentation happen in the Confluence Clojure space.
  • Issue tracking happens on the the Clojure JIRA site.
  • This documentation is maintained in the gh-pages branch of clojure on GitHub and is always available online GitHub pages for Clojure. If you wish to have a version for off-line use you can use the download button on the gh-pages branch page at GitHub.
